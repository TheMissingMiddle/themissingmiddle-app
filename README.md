# The Missing Middle - App


> A quick prototype with [Twilio](https://www.twilio.com) and [Ionic](https://ionicframework.com)


## Getting Started

1. Make sure you have [Ionic](https://ionicframework.com) installed.
2. Install your dependencies
    
    ```
    npm install
    ```

3. Start your app
    
    ```
    ionic serve
    ```
